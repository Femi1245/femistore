# Zumelia

Premium social app for conversation, live streaming, business, and community.

## Web

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

Production: [https://itunes-mu.vercel.app](https://itunes-mu.vercel.app)

## Android APK

Install guide: [/download](https://itunes-mu.vercel.app/download)

CI builds the Capacitor APK on pushes that touch `android/` or `capacitor.config.ts`.
